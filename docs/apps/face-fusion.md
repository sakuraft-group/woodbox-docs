# 人脸融合

## 什么是人脸融合

对两张人脸进行人脸融合操作，并返回融合后的图片，生成的人脸同时具备两张人脸的外貌特征。

## 操作提示

上传的模版图人像(第一张)部分应距离边缘有一定距离。

上传的目标图人像(第二张)部分应在图片正中正脸清晰。

上传的图片分辨率不能超过1920*1080。

上传的图片支持jpg/png/bmp格式。
